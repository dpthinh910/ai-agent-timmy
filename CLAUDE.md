# ai-agent-timmy

Consolidated AI agent workspace — skills, hooks, and commands for building intelligent coding workflows.

## Stack
- **AI Skills**: 26 curated AI-focused skills (agentic engineering, autonomous loops, deep research, etc.)
- **Agents**: 8 specialized subagents (planner, architect, code-reviewer, loop-operator, etc.)
- **Commands**: 14 slash commands for planning, evaluation, multi-agent orchestration
- **Hooks**: Session lifecycle, code quality, continuous learning, cost tracking

## Quick Commands
| Command | Purpose |
|---------|---------|
| `/plan` | Implementation planning |
| `/verify` | Run verification loop |
| `/learn` | Extract patterns mid-session |
| `/multi-plan` | Multi-agent task decomposition |
| `/multi-execute` | Multi-agent orchestration |
| `/orchestrate` | Multi-agent coordination |
| `/evolve` | Cluster instincts into skills |

## Contexts
- `dev.md` — Development mode
- `review.md` — Code review mode
- `research.md` — Research/exploration mode
